# Input parameters guide

**Reference:** Run 4 — 4 A, 150 µs, 80 %

## Pattern
- Low current (4 A)
- Long pulse-on (150 µs)
- High duty (80 %)
- Low removal (~0.5)
- Fine servo feed (tool/pore = 3.82)

## Extra inputs (not in 16-run table)
1. Gap voltage — stable finish
2. Servo feed — very fine
3. Dielectric flush — continuous
4. Electrode dress — 900 µm
5. Phase 2: Tool_x, Tool_y from (0,0)

## Avoid
- I ≥ 8 A, T ≤ 75 µs, removal > 0.9
